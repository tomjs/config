# tomjs

[English](./README.md) | **中文**

> 前端项目代码规范整理，方便快速搭建项目

## 使用

以 `pnpm` 为例

### 通用

```bash
pnpm add -D eslint prettier @tomjs/eslint @tomjs/prettier @tomjs/tsconfig
```

### commitlint

```bash
pnpm add -D husky lint-staged @commitlint/cli @tomjs/commitlint
```

### style

```bash
pnpm add -D stylelint @tomjs/stylelint
```
