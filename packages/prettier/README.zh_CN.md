> 前端项目 [prettier](https://prettier.io/) 代码格式化规范配置，推荐 `node>=16`。

## 使用

- 安装依赖

```bash
pnpm add -D prettier @tomjs/prettier
```

- 修改 `prettier.config.{js,ts,mjs,mts}` 配置

```js
export default '@tomjs/prettier';
```
